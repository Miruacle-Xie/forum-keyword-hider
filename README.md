# Forum Keyword Hider Pro

一款轻量、强大的论坛内容过滤工具。不仅能按文本过滤，更支持深度 HTML 结构匹配，助你回归清爽的阅读体验。

### 核心功能

1.  **分路过滤系统**：支持“文本匹配”与“代码属性匹配”双路并行，精准打击，杜绝误伤。
2.  **灵活的匹配策略**：每一条规则均可独立选择“普通包含”或“正则表达式”模式。
3.  **实时生效机制**：告别“保存”按钮，所有勾选、删除或编辑操作均即时自动保存并同步至页面。
4.  **智能容器隐藏**：自动识别论坛列表结构（`tr`/`tbody`/`li`），隐藏时彻底抹除相关 DOM 节点，不留空白行。
5.  **动态兼容**：全面支持异步加载（AJAX/无限滚动）的帖子列表。
6.  **通用图片防放大**：内置通用事件拦截逻辑，无需绑定特定网站类名，一键禁用全站缩略图悬停预览。
7.  **内页保护**：支持自定义网址黑名单，确保进入帖子详情页后过滤功能自动静默，防止误隐重要信息。

---

### 使用指南

* **规则管理**：
    * **匹配文本**：适用于隐藏标题中包含特定词汇或特定内容的帖子。
    * **匹配代码**：适用于通过 `id`、`class` 等 HTML 属性进行隐藏（如：输入 `[class*="topicrow"]` 或 `id="stickthread_\d+"`）。
* **正则模式**：当选择“正则”时，每一行输入框将作为标准的 JavaScript 正则表达式进行匹配。
* **开关控制**：左侧勾选框用于快速启停单条规则，无需删除规则即可暂时关闭。
* **图片拦截**：勾选“禁止图片悬停放大”即可全局禁用图片预览，插件将自动拦截悬浮窗触发事件。

---

### 提示

* 不同论坛的 HTML 结构差异较大，通用拦截规则虽不绑定特定类名，但若遇到特殊设计，请优先使用“代码匹配”中的 `[class*="..."]` 选择器进行精准清理。
* 插件已内置默认黑名单（包含 `viewthread`、`thread-` 等关键字），确保阅读体验连贯。

---

### 安装步骤

1.  打开 Chrome 浏览器，访问 `chrome://extensions/`。
2.  开启右上角的 **“开发者模式”**。
3.  点击左上角的 **“加载已解压的扩展程序”**。
4.  选择存放本插件代码的文件夹即可加载。
